## Instalacion
Clona el repositorio, debe tener npm instalado en su pc y dentro del proyecto ejecuta el comando
```bash
npm install
```
Recuerde que en la carpeta 
```bash
assets/media
```
tiene que colocar un archivo mp4 con el nombre
```bash
video.mp4
```
